# 🌸 Bloomly

Bloomly is a flower arrangement booking and management web application built with Laravel.

Users can browse available floral arrangements for different occasions — weddings, birthdays, and more — and book a date for their desired decoration service. Admins manage all bookings through a calendar dashboard, confirming or cancelling requests with ease.

## Features

### User Side
- Register & Login
- Browse flower arrangements by occasion type
- Book a date for a floral decoration service
- View booking status (Pending / Confirmed / Cancelled)
- Cancel own booking

### Admin Side
- Login as Admin
- Add, Edit, Delete flower arrangements
- View all bookings on a Calendar dashboard
- Confirm or Cancel bookings

## Tech Stack
- Laravel (PHP)
- MySQL
- Blade templates
- Laravel Breeze (Authentication)
